# picar
A DIY car based on Raspberry Pi 3 and nodejs.

代码运行在Raspberry Pi 3机器上，需要升级Raspberry Pi系统的nodejs。

```
npm install
npm start
```

可以通过手机或者电脑访问web server的小车控制页面，远程控制小车。
